About Block Farm Editor

What it is

Block Farm Editor is an Umbraco package designed to make building and maintaining block-based pages faster, more consistent, and more editor-friendly.

It focuses on a practical workflow: define blocks, compose pages visually, and reuse layouts to keep content consistent across your site.

Core capabilities

  • Composable blocks to build pages from reusable pieces
  • Layouts to save and reapply common block structures
  • Import/Export to move configurations between environments

Why it was built

The idea for Block Farm Editor came from a simple question: “Could I build a visual editor in Umbraco that’s easily extensible while providing a fully integrated visual experience?” As such, focus was placed on the editor experience and extensibility from the very beginning.
